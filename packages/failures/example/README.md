An example of using [failures] and [failures_builder] packages.

In order to run it:
1. re-create missing platform-specific parts of Flutter project by running: `flutter create .`
2. run [build_runner] to generate failures: `dart run build_runner build`
3. run the example app itself: `flutter run .`

[failures]: https://pub.dartlang.org/packages/failures
[failures_builder]: https://pub.dartlang.org/packages/failures_builder
[build_runner]: https://pub.dev/packages/build_runner
